# Portal Berita Indonesia

## Overview
Sebuah website sederhana yang menampilkan berita dari negara indonesia. 
Api yang digunakan adalah dari [News Catcher API](https://newscatcherapi.com/).


## Project setup
```
npm install
```

npm run serve
